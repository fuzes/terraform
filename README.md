## terraform 설치
```
brew install terraform
```

## create terraform.tfvars.json file 
```json
example :
{
    "aws_access_key":"ABCDEFGASDFASDFSDFSFASF",
    "aws_secret_key":"sfksfi3fskfjdskfi3fkdfjsdjf3i",
    "aws_region":"ap-northeast-2"
}
```

## terraform aws plugin install
```
terraform init
```

## terraform apply
```
terraform apply
```

## terraform destroy
```
terraform destroy
```
